## Features

- SignIn/SignUp(Google Authentication / Email,Password )using Firebase
- Multiple gaming system
- Play Game (Tik Tak Toe , Snake , Quiz , Calculator )
- Using Drawer , Fragment , Recycleview 
- Firebase Authentication
